      OUTPUT_PATH = Path("docs/data/results.json")

      def write_json(df: pd.DataFrame) -> None:
          if len(df) < 100:
              log.error(
                  f"Only {len(df)} rows produced — aborting JSON write (minimum 100 required)"
              )
              sys.exit(1)
          OUTPUT_PATH.parent.mkdir(parents=True, exist_ok=True)
          rows = df.where(df.notna(), other=None).to_dict(orient="records")
          payload = {
              "generated_at": datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ"),
              "rows": rows,
          }
          OUTPUT_PATH.write_text(
              json.dumps(payload, separators=(",", ":")),
              encoding="utf-8",
          )
          log.info(f"Results written to {OUTPUT_PATH} ({len(rows)} rows)")

    Key details:
      - df.where(df.notna(), other=None) converts NaN/NaT to Python None, which serializes
        to JSON null (avoids invalid JSON NaN literals)
      - sys.exit(1) satisfies PY-03: non-zero exit, write is skipped entirely
      - OUTPUT_PATH.parent.mkdir(parents=True, exist_ok=True) creates docs/data/ at runtime
        if it does not exist
      - separators=(",", ":") satisfies PY-04 compact encoding
      - generated_at satisfies PY-02 ISO timestamp requirement

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| screener → filesystem | write_json() writes user-controlled data to docs/data/results.json |
| filesystem → git | Actions commits results.json; default GITHUB_TOKEN scoped to repo |
| git → Pages | GitHub Pages serves committed files publicly with no auth |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | results.json content | accept | Data originates from yfinance/Finnhub APIs already validated by pipeline; no user input path |
| T-02-02 | Denial of Service | write_json() row guard | mitigate | sys.exit(1) on < 100 rows prevents empty or near-empty file from being committed |
| T-02-03 | Information Disclosure | Pages URL is public | accept | Intentional — public dashboard is the project goal; no PII or credentials in results.json |
| T-02-04 | Elevation of Privilege | GITHUB_TOKEN write scope | accept | Set in Phase 1; scoped to this repo only; no cross-repo access |
| T-02-SC | Tampering | No new package installs in this phase | accept | No npm/pip/cargo installs; json, sys, pathlib, datetime are Python stdlib |
</threat_model>
